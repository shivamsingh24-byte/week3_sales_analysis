

# Sales Data Analysis Report

## Project Overview
This project analyzes sales data using Python and the Pandas library.

## Objectives
Load a dataset from a CSV file, explore the dataset, clean the data, and calculate important sales metrics.

## Dataset
File name: sales_data.csv

Columns:
Product
Quantity
Price

## Analysis Steps
Load dataset using pandas.
Check dataset information.
Remove duplicate rows.
Handle missing values.
Create Total_Sales column.
Calculate revenue and statistics.

## Metrics Calculated
Total Revenue
Best Selling Product
Average Sale Value
Highest Sale
Lowest Sale

## Conclusion
This project demonstrates how Python and pandas can be used to analyze sales data and generate useful insights.
